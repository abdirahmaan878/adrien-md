## 🔑 Pairing Dashboard

> Securely link your device using the official ADRIEN-MD pairing portals below: if connection issues persist, VPN activation may help resolve them.

### 📱 Pair via Terminal (CLI Method)

```bash
# Start the bot in pairing mode
npm start -- --pair

# Or via environment variable
PAIRING_NUMBER=2348012345678 npm start
```

> Replace the number above with your **full international WhatsApp number** — no `+`, no spaces, no dashes.

Once started, you'll see something like this in the terminal:

```text
╔══════════════════════════════════╗
║   ADRIEN-MD PAIRING CODE         ║
╠══════════════════════════════════╣
║                                  ║
║      🔑  ABCD-EFGH               ║
║                                  ║
║   Code expires in 60 seconds     ║
╚══════════════════════════════════╝
```

### 🔗 Link to WhatsApp

1. Open **WhatsApp** ➜ tap **⋮** menu (Android) or **Settings** (iOS)
2. Tap **Linked Devices** ➜ **Link a Device**
3. Tap **Link with phone number instead**
4. Enter the **8-character code** shown in your terminal
5. ✅ Adrien-MD is now connected!

> 💡 **Tip:** Your session is stored in the `auth/` folder. Back it up if you don't want to re-pair later.

<details>
<summary><b>❗ Pairing Troubleshooting (click to expand)</b></summary>

| Issue | Fix |
|-------|-----|
| `Code expired` | Restart the bot to regenerate a fresh code |
| `Number not registered` | Ensure WhatsApp is active on that number |
| `Connection failure` | Delete the `auth/` folder and pair again |
| `Logged out instantly` | Remove unused linked devices in WhatsApp |
| `Code not appearing` | Ensure Node.js is **v20 or higher** |

</details>

## ⚙️ Environment Variables

| Variable | Description | Default |
|----------|-------------|---------|
| `SESSION_ID` | Your Adrien-MD session ID | *required* |
| `PREFIX` | Command prefix | `.` |
| `OWNER_NUMBER` | Your WhatsApp number | *required* |
| `BOT_NAME` | Display name of the bot | `Adrien-MD` |
| `LANGUAGE` | Default response language | `en` |
| `MODE` | `public` or `private` | `public` |
| `AUTO_READ` | Auto-mark messages as read | `false` |
| `AUTO_REACT` | Auto-react to messages | `false` |
| `ANTI_DELETE` | Recover deleted messages | `true` |
| `ANTI_CALL` | Auto-reject calls | `false` |

## 📂 Project Structure

```text
Adrien-MD/
├── 📁 assets/          # Images, fonts & media
├── 📁 auth/            # Saved sessions (auto-generated)
├── 📁 commands/        # Plugin-based command modules
│   ├── admin/
│   ├── downloader/
│   ├── fun/
│   ├── group/
│   └── owner/
├── 📁 lib/             # Core utilities & helpers
├── 📁 config/          # Bot configuration
├── 📁 database/        # Local DB & schemas
├── 📄 index.js         # Entry point
├── 📄 config.js        # Main config file
├── 📄 package.json
└── 📄 .env.example
```

## 🎯 Command Examples

| Command | Description |
|---------|-------------|
| `.menu` | Show full command list |
| `.ping` | Check bot latency |
| `.sticker` | Convert image/video to sticker |
| `.play <song>` | Download YouTube audio |

| `.tagall` | Mention everyone in a group |
| `.kick @user` | Remove a user from group |
| `.antilink on` | Enable anti-link protection |
| `.restart` | Restart the bot |
